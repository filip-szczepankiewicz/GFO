# GFO — Geometric Filter Optimization for Rotation Sampling
A tool for diffusion MRI.  Implements methods for designing rotation sets that enable accurate powder averaging for triaxial diffusion tensors, as described in https://arxiv.org/abs/2601.14970. Work by Sune Jespersen and Filip Szczepankiewicz. 

Tested on Matlab 2025b, requires 'Optimization Toolbox' and 'Parallel Computing Toolbox'. 

To get started, look at 'example.m'.

Usage is free, but please cite the paper.
